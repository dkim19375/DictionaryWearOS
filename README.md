# Dictionary
A free and open source dictionary app for Wear OS.

Tested on Galaxy Watch 5.

## Features
- Can view:
  - Parts of speech
  - IPA phonetics
  - Examples usages
  - Synonyms
  - Antonyms
- Supports saving words offline (otherwise, internet connection is required)
