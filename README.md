# Dictionary
A free and open source dictionary app for Wear OS.

## Features
- Can view:
  - Parts of speech
  - IPA phonetics
  - Example usages
  - Synonyms
  - Antonyms
- Supports saving words offline (otherwise, an internet connection is required)
